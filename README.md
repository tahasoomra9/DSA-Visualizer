# DSA Viz

DSA Viz is an ongoing learning project for Data Structures and Algorithms.
The goal is to help users understand famous DSA patterns through interactive visual diagrams and step by step execution.

Right now, the project is focused on core visualization experiences.
Over time, it will grow into a full learning platform with richer algorithm coverage and code level debugging views.

## Vision

The long term direction of DSA Viz is:

- Teach famous DSA patterns in a visual and practical way
- Show algorithm iterations clearly, including what happens at each step
- Add a Monaco style code editor with debugger style line by line execution
- Let users paste their own code and generate a visualization from it

Before the full custom code visualization feature is ready, this site serves as a focused DSA visualization platform.

## Current Status

- Project is in active development
- Basic visualizer flow is available
- Users can set array and target input
- Step controls allow moving through algorithm execution

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser:

```text
http://localhost:3000
```

## Roadmap

1. Add more famous DSA patterns and problem types
2. Improve diagram quality and interaction depth
3. Build Monaco style editor integration
4. Add debugger style iteration playback for each code line
5. Support user pasted code to auto generate visual explanations
